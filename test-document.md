# Test Document for AI Chatbot

This is a test document to demonstrate the file upload and document analysis capabilities of the AI chatbot application.

## Features Demonstrated

### 1. File Upload
- Drag and drop functionality
- File type validation (PDF only)
- File size validation (10MB limit)
- Progress tracking during upload
- Error handling and user feedback

### 2. Document Processing
- PDF text extraction
- Content chunking for analysis
- Vector embeddings generation
- Storage in MongoDB database
- Integration with vector search

### 3. AI Integration
- Document-based question answering
- Retrieval Augmented Generation (RAG)
- Context-aware responses
- Source attribution

### 4. User Experience
- Real-time upload progress
- Success/error notifications
- Document management interface
- Search within documents

## Technical Implementation

### Backend Features
- Multer for file handling
- PDF-parse for text extraction
- OpenAI embeddings for vector search
- MongoDB for document storage
- Express.js API endpoints

### Frontend Features
- React components for file upload
- Drag and drop interface
- Progress indicators
- Error handling
- Responsive design
